# 🛒 TecnoKaijin E-commerce

Tienda online de tecnología desarrollada con React y Node.js

## 🚀 Instalación

### Backend
```bash
cd backend
npm install
npm start
```

El servidor correrá en: http://localhost:5000

### Frontend
```bash
cd frontend
npm install
npm start
```

El frontend correrá en: http://localhost:3000
